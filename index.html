HA DINH'S PORTFOLIO TUTORIAL

Description

The portfolio is made using markdown, HTML5, CSS3, and ES6. Screenshot of the entire page is attached below.

Instructions

Step 1: Make a new folder and open it in Visual Studio Code or your favourite IDE
Step 2: Fork your own copy of rhildred/browserBlog to your account
Step 3: Copy the HTTPS url under the Code button
Step 4: Open a new terminal and run "git clone <copied url here> ."
Step 5: Run "npx http-server" (you will need to install node.js for this to work)
Step 6: Surf to http://localhost:8080
Step 7: Add, edit, delete, and generally customize the files in the folder until you are satisfied with the aesthetic and the functionality of the site. Don't forget to highlight your skills and show your previous projects
Step 8: In a new terminal, type in "git add -A" to add new files, add new changes, and remove files
Step 9: Type in "git commit -m "uploaded"" to commit changes
Step 10: Wait for the previous command to finish, then type in "git push origin master --force" to send your committed changes to the remote repository
Step 11: Click on the Settings tab on your repository
Step 12: Scroll down to GitHub Pages, select the master branch under Source, then hit Save
Step 13: Congrats! Your portfolio is now published at <your username>.github.io/<project name>
